# test-md5-tools 

> Verify common crypto libraries to check correction of building MD5 hash

## Install

```
$ npm install test-md5-tools
```

## How to add a new library?

1. Create `*.js` file
2. Module should export method `encode`
3. Import module info `index.js` file
4. Run tests
